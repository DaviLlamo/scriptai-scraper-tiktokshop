### TikTok Shop Trend Analyzer + Gerador de Roteiros com IA

Desenvolvi uma aplicação desktop em Python capaz de automatizar a identificação de tendências no TikTok Shop e gerar roteiros otimizados para vídeos virais utilizando inteligência artificial.

A aplicação realiza scraping de produtos e vídeos em alta, organiza os dados em planilhas Excel e utiliza modelos de IA para criar roteiros estratégicos com foco em conversão, incluindo gatilhos de engajamento, estrutura de vídeo e chamadas para ação.

Além disso, o sistema gera automaticamente relatórios em PDF contendo os roteiros produzidos, facilitando o uso para criadores de conteúdo e operações de dropshipping.

O projeto conta com interface gráfica (GUI), automação completa do fluxo de dados e integração com APIs de IA.

# Principais tecnologias:

- Python
- Playwright (web scraping)
- Tkinter (interface gráfica)
- OpenPyXL / Pandas (manipulação de dados)
- ReportLab (geração de PDF)
- API de IA (Gemini)

## 🔥 Funcionalidades

- 🔎 Buscar vídeos do TikTok Shop por palavra-chave
- 📈 Coletar tendências do dia (trending)
- 📊 Gerar automaticamente arquivos Excel com os dados
- 🤖 Criar roteiros virais com IA
- 📄 Gerar relatórios em PDF com os roteiros
- 🖥️ Interface gráfica simples (Tkinter)

---

## 🧠 Como funciona

1. O scraper coleta até 5 vídeos do TikTok Shop
2. Os dados são salvos em um arquivo Excel
3. O usuário seleciona o Excel
4. A IA gera 2 roteiros por vídeo
5. O sistema pode gerar um PDF com todos os roteiros

---
